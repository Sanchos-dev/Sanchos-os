# sanchosctl

`sanchosctl` is the command-line control plane for sanchos-os.

## Current commands

- `system info`
- `system doctor`
- `profile list`
- `profile info <name>`
- `profile apply <name>`
- `module list`
- `module info <name>`
- `module enable <name>`

## Packaging status

This package source is still minimal and should be treated as project scaffolding.
