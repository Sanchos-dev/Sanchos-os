# sanchos-os v0.1 target

## Goal

Ship a usable first milestone that proves the product shape:

- Debian 12 base
- KDE Plasma desktop
- branded login and wallpaper set
- first-boot setup flow
- KVM/libvirt working out of the box in the `desktop-virt` profile
- NekoBox preinstalled for desktop profiles
- `sanchosctl` able to inspect and perform basic host actions
- basic control center shell

## Success criteria

A tester should be able to:

1. install or bootstrap a clean Debian system
2. log into a branded Plasma session
3. run first-boot setup
4. create a VM from an ISO using `sanchosctl`
5. start and stop that VM
6. use NekoBox without hunting for a VPN client

## Deliverables

### Platform
- `sanchos-core` package scaffold
- `sanchos-desktop` package scaffold
- `sanchos-desktop-virt` package scaffold
- `sanchos-branding` package scaffold
- `sanchos-theme` package scaffold

### Tools
- `sanchosctl` with system/profile/module/vm commands
- `virt-install` wrapper for VM creation
- host health checks

### Desktop
- Plasma defaults
- SDDM theme scaffold
- wallpapers
- first-boot wizard
- control center shell

## Out of scope

- clustering
- HA
- live migration
- custom installer UI
- own package repository infrastructure
- replacement for virt-manager in v0.1
