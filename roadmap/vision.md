# Vision

sanchos-os is a Debian-based platform for people who want one machine to behave like both a polished workstation and a local infrastructure host.

The system should feel coherent from boot to desktop: good defaults, solid visuals, predictable tooling, native virtualization, and modular integration points for external services.

The long-term aim is not to clone Proxmox on a desktop. It is to make local virtualization, containers, service access, and host control feel native in a real daily-use operating system.
