# Packaging

The project starts with standard Debian packaging conventions.

## Early approach
- use apt and dpkg
- keep project-specific logic in dedicated packages
- use meta-packages for install roles

## Planned package families

- sanchos-branding

- sanchosctl
- sanchos-control-center
- sanchos-desktop
- sanchos-desktop-virt
