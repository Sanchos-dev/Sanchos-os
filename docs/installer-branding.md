# Installer and live image branding

The current ISO path still builds through Debian `live-build`, so v9 focuses on making the **live session** feel like sanchos-os rather than pretending there is already a fully custom installer stack.

## What v9 does

- installs the purple-first wallpaper collection into the image
- applies the purple visual preset on first login
- ships the purple SDDM theme scaffold
- keeps the live session much closer to the final installed system

## What still comes later

- deeper bootloader splash branding
- a fully branded graphical installer flow
- tighter Calamares/live-installer customization once the ISO path is more stable
